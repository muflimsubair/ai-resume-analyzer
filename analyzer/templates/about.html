🌿 About AI Resume Analyzer

This project is an AI-powered Resume Analyzer built using Django and Python. It uses Natural Language Processing (NLP) techniques to compare resumes with job descriptions and calculate a match score.

The system extracts text from uploaded resumes and analyzes it using CountVectorizer and Cosine Similarity to evaluate how well a candidate fits a job role.

🛠 Technologies Used:
Python Django HTML CSS Bootstrap NLP Scikit-learn
